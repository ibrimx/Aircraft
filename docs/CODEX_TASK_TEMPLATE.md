Task Structure:
كل task لازم يحتوي:
1) CONTEXT — ما الملفات التي يقرأها Codex قبل البدء
2) TASK — ما المطلوب كتابته (ملف واحد أو ملفين)
3) RULES — القواعد المحددة لهذا الـ task
4) SELF-CHECK — قائمة مراجعة قبل التسليم
